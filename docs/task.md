## 技术栈建议

这些项目通常使用以下技术组合：

```
核心：HTML5 + CSS3 (Tailwind/Bootstrap) + Vanilla JS
增强：Vue 3 (CDN 版) / React (CDN 版) / Alpine.js
存储：localStorage / IndexedDB / URL Hash 参数
图表：Chart.js / ECharts (CDN)
编辑：CodeMirror / Monaco Editor (CDN)
部署：GitHub Pages / Vercel / Netlify / Cloudflare Pages
```
